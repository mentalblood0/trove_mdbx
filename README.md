# 🏴‍☠️ trove_mdbx

Arbitrary JSON-structured data, all-indexing storage backed with [libmdbx](https://libmdbx.dqdkfa.ru/)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     trove:
       github: mentalblood0/trove_mdbx
   ```

2. Run `shards install`
